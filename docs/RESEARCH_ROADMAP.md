# Research Roadmap: Lutufi

## Phase 1: Deep Theory (Pre-Coding)
- [ ] **Mathematical Audit:** Finalize the unification approach for DAG semantics on non-DAG social structures.
- [ ] **Literature Review:** Complete comparative analysis of `pgmpy`, `bnlearn`, and `NetworkX`.
- [ ] **Requirement Analysis:** Map the specific failures in current research workflows (e.g., "What currently breaks?").

## Phase 2: Core Architecture (Foundation)
- [ ] **Data Representation:** Design sparse matrix storage for large-scale probabilistic tables.
- [ ] **Inference Engine Alpha:** Implement exact inference for static, small-scale networks.
- [ ] **Python Bindings:** Establish the first iteration of the user-facing API.

## Phase 3: The Example Library (Validation)
- [ ] **Epidemiology Module:** Build SIR/SEIR models with probabilistic transmission.
- [ ] **Financial Risk Module:** Build interbank contagion models.
- [ ] **Influence Mapping:** Build models for belief propagation in social groups.

## Phase 4: Scaling & Dynamics (Growth)
- [ ] **Dynamic Networks:** Implement temporal updates and time-stepping.
- [ ] **Parallelization:** Optimize for multi-core and potentially GPU-accelerated inference.
- [ ] **Serialization:** Create the formal "Lutufi Model" file format.

## Phase 5: Ecosystem & Credibility
- [ ] **JOSS Submission:** Prepare and submit the introductory paper.
- [ ] **Simulation Module:** Add forward-time simulation capabilities.
- [ ] **Visualization:** Develop a semantically aware visualization suite.

## Continuous Tasks
- **Documentation:** Weekly updates to theory and API docs.
- **Security & Ethics:** Ongoing review of misuse potential and adversarial robustness.
- **Community Engagement:** Building a base of academic and institutional collaborators.
